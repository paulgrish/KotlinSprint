✅ 2: Арифметика | Задачи

Почему я рекомендую придерживаться принципа 1 урок - 1 файл в рамках обучения?

В реальной разработке нужно избегать копирования кода, но в данном случае для обучения так будет нагляднее. А в рамках нашего обучения — это облегчит процесс код-ревью. Таким образом я смогу проверять и оценивать решения по отдельности. Подробнее вся схема работы описана в инструкции по сдаче заданий в основном разделе с задачами.

Каждый урок создается в отдельном пакете, каждая задача создается в отдельном файле. Даже если в условиях будет сказано дописать или расширить предыдущую задачу, ее решение необходимо будет копировать в новый файл.

Не каждый, кто даже уже знаком с основами программирования, умеет писать правильно оформленный production-ready код и работать с git так, как это делается абсолютно во всех компаниях. Так что ты по адресу и все мои правки (даже по пробелам) пойдут на пользу — просто доверься, а понимание обязательно придет.  В следующих уроках я расскажу еще немного про историю git и в чем его концепция на примере.
