Добро пожаловать на задачи первого урока!

Надеюсь, ты не упустил из внимания документ с инструкцией по работе с git. Можешь потренироваться “на кошках” перед сдачей задач, повторяя шаги по видео и закрепляя подробности в текстовом описании шагов. Ошибаться, пересоздавать проект, если все сломалось с первого раза не страшно и даже полезно — мы здесь, чтобы учиться.

Задачи первого урока на правильное создание и нейминг переменных. Уверен, не должно возникнуть серьезных проблем. Отличная возможность уделить внимание настройке среды разработки и git.

И не забывай про самое главное – новая ветка всегда создается из ветки master, который нужно предварительно обновить. Помни, что в пулл-реквест с конкретной одной задачей не должны попадать измененные файлы для других задач. В следующих уроках еще раскрою эту тему подробнее, с примерами. И самое главное начать — погнали!
